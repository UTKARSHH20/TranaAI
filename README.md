## ⚡ What is TranaAI?

</div>

**TranaAI** *(from Sanskrit* **त्राण** *— Rescue · Deliverance · Protection)* is an AI-powered disaster response command center built for **Google Solution Challenge 2026**. It gives emergency coordinators real-time situational awareness across **30 disaster zones** in India — combining live weather data, seismic monitoring, and Google Gemini AI to deliver actionable intelligence when every second matters.

<div align="center">

```
┌─────────────────────────────────────────────────────────────────────┐
│  🚨 Disaster Strikes  →  📡 TranaAI Activates  →  🤖 Gemini Thinks  │
│  🗺️ Map Updates       →  👥 Volunteers Deploy  →  ✅ Lives Saved     │
└─────────────────────────────────────────────────────────────────────┘
```

## 🧮 Priority Scoring Algorithm

</div>

The core allocation engine uses a **weighted composite score**:

```
P(zone) = w₁ × (severity / 10) + w₂ × min(pop / 400K, 1) + w₃ × min((20 - res) / 20, 1) + w₄ × min(max(trend, 0) / 2, 1)
```

<div align="center">

| Weight | Default | Factor | Rationale |
|:------:|:-------:|--------|-----------|
| **w₁** | `0.40` | Base Severity | Ground truth conditions |
| **w₂** | `0.20` | Population | Human impact scale |
| **w₃** | `0.20` | Resource Gap | Supply deficit urgency |
| **w₄** | `0.20` | Severity Trend | Rate of deterioration |

> 🎛️ All weights are **user-tunable in real time** via the Priority Engine panel

</div>

## 🔥 Multi-Hazard Risk Scoring

</div>

<div align="center">

| 🌊 Flood Risk | 🌀 Cyclone Risk | 🌡️ Heatwave Risk | 🌍 Earthquake Risk |
|:-------------:|:---------------:|:-----------------:|:------------------:|
| **40%** weight | **25%** weight | **20%** weight | **15%** weight |
| Precipitation + probability | Wind speed + storm codes | Temperature > 42°C | USGS events within 200km |

</div>

> Uses the **Haversine formula** for geographic distance calculations between cities and seismic events.

## 🚀 Getting Started

</div>

### Prerequisites

```bash
Node.js ≥ 18  ·  pnpm (recommended)  ·  Gemini API Key  ·  Google Maps API Key
```

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/UTKARSHH20/trana-ai.git
cd trana-ai

# 2. Install dependencies
pnpm install

# 3. Configure environment (optional — keys can also be entered in-app)
echo "VITE_GEMINI_API_KEY=your_gemini_key_here" > .env
echo "VITE_GOOGLE_MAPS_KEY=your_maps_key_here" >> .env

# 4. Start development server
pnpm dev
# → http://localhost:5173/
```

<details>
<summary><b>📦 Environment Variables Reference</b></summary>

<br/>

| Variable | Required | Description |
|----------|:--------:|-------------|
| `VITE_GEMINI_API_KEY` | Optional | Google Gemini API key (can enter in-app) |
| `VITE_GOOGLE_MAPS_KEY` | Optional | Google Maps JavaScript API key |

> **Note:** Both keys can be entered directly in the UI. They're stored only in browser memory — never sent to any backend.

</details>
